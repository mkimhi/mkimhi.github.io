Overview

Deep Neural Lesion (DNL) exposes a severe vulnerability in DNNs: flipping only a handful of sign bits in model weights can catastrophically disrupt performance across vision and language domains. Our data-free, optimization-free approach requires only write access to stored parameters.

  • ResNet-50: 2 flips → 99.8% accuracy drop (76% → 0%)
  • Mask R-CNN / YOLOv8-seg: 1-2 flips collapse detection/segmentation
  • Qwen3-30B (MoE LLM): 2 flips reduce reasoning accuracy from 78% → 0%

Methodology

DNL Attack Variants

Pass-Free DNL: Identifies critical parameters using magnitude-based heuristics and early-layer targeting. No additional computational passes required.

Enhanced 1-Pass DNL: Refines parameter selection with a single forward and backward pass on random inputs:

Find parameters that maximize gradient-weighted sensitivity with minimal computation

Why Sign-Bit Flips?

  • Cleanest disruption: flipping the sign bit instantly negates weight values
  • Hardware alignment: bit flips in fixed positions are more feasible in real attacks (Rowhammer, DMA, firmware exploits)
  • High-magnitude early-layer weights are most critical for feature propagation
Flipping a single sign bit in an early-layer edge detector kernel transforms learned features, corrupting downstream representations and severely impairing recognition.

Defense & Implications

We propose selective hardening of critical parameters. By protecting only the most vulnerable weights, models achieve substantial resilience without major performance overhead.

This work demonstrates that once attackers gain parameter write access (via firmware exploits, rootkits, or hardware vulnerabilities), minimal computation suffices to inflict catastrophic model failure. The data-free nature makes detection and mitigation exceptionally challenging.
